## 2025-03-06 - Safe Component Extraction for Performance
**Learning:** Extracting list items to `React.memo` components is a common performance optimization for lists with interactive elements (like accordions or hovers). However, when modifying existing files, be careful not to introduce regressions by deleting or modifying unrelated data mappings (`map()`) simply because they look like "dead code", especially in files where the render function heavily depends on the data shape. The strict rule of "ONE small performance improvement" means touching a second file and breaking its data fetching/mapping violates both scope and safety.
**Action:** When acting as Bolt, strictly stick to one clear, isolated optimization in a single file unless a second file absolutely requires a paired change. Before deleting "dead code" in React components, verify where that variable is used in the JSX tree.
